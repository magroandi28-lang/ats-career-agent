# ATS-Career Agent 🤖
**MI-alapú álláskeresési asszisztens**

## Mit csinál?
Két MI-rendszer egy helyen — leveszi a válladról az álláskeresés terhét:

### 1. ATS-Hacker (CV Optimalizáló)
- Beolvassa a PDF önéletrajzot és az álláshirdetést
- Elemzi a kulcsszavakat és az egyezéseket
- Javaslatot tesz a CV módosítására hogy az MI-szűrők ne dobják ki

### 2. Személyi Karrier-Ügynökség (CrewAI ágensek)
- **Kutató ágens** → állásokat keres (Profession, Jobline, Indeed)
- **Elemző ágens** → cégkutatást végez
- **Író ágens** → személyre szabott motivációs levelet ír

## Technológia
- CrewAI · Claude API · pdfplumber · Streamlit · Render
- Apify · SerpAPI · Profession RSS · Jobline RSS
